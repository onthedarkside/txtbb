Language Install Instructions

Welcome to PBLang, the international bulletin board system with many exciting features!

Please reade the !!!!!README-FIRST!!!!!-file before installation. If you want to add another language, please read this very carefully:

  1. Make sure nobody else is working on the language you plan to work on, by checking the PBLang-support forum. If there is someone working on your language, ask if you may help.
  2. To add another language, use the templates/pb/language/lang_temp.php file.
  3. Copy this file to a new file, giving it the same name, but replacing the letters "temp" with the two letter country-code as used in ISO of the country whose language you want to translate it to. Then make this file writeable, chmod 666.
  4. Open this file, you can do that later from within PBLang's admin center, and add all strings, as described in this file.
  5. Send that file to me at
    this address for inclusion in the distribution package.
  6. Include your file in the templates/pb/language-folder.
  7. Now you should be able to see this language in your language selection list of the PBLang board.
  8. It's good to have buttons also in your language. To create some, either get a friend or do it yourself - there is also a person on the support forum who is willing to create buttons for you - check the language section there.
  9. To keep your language file up-to-date, I would like to have you in the developers' mailing list. Please contact me for that reason. The mailing list gives you access to all the latest program files.
